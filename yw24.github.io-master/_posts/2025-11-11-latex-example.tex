\documentclass{article}
\usepackage{amsmath}

这是在 Jekyll 中使用 LaTeX 的一个示例。

我们可以使用 `jekyll-pandoc` 插件来渲染 `.tex` 文件。这使得我们可以在博客文章中直接使用 LaTeX 语法。

\section{数学公式}

下面是一些数学公式的例子。

\subsection{行内公式}

爱因斯坦的质能方程是 $E=mc^2$。这是一个行内公式。

\subsection{块级公式}

欧拉恒等式被誉为数学中最美的公式之一：
$$ e^{i\pi} + 1 = 0 $$

这是一个块级公式。

\section{列表}

我们也可以创建列表：
\begin{itemize}
    \item 列表项一
    \item 列表项二
    \item 列表项三
\end{itemize}

\section{总结}

通过 Pandoc，Jekyll 可以很好地支持 LaTeX，这对于需要撰写大量数学内容的博客来说非常方便。
