在Ubuntu系统中截图、录屏的方法，包括使用系统快捷键和Kazam。

<!-- more -->

## 使用系统快捷键

### 截图

| 功能 | 快捷键 |
| :---- | :---- |
| 对于桌面截图 | `PrtSc` |
| 对于窗口截图 | `Alt`+`PrtSc` |
| 对于选定区域截图 | `Shift`+`PrtSc` |

### 录屏

使用`Ctrl`+`Alt`+`Shift`+`R`开始和结束录制，默认保存在`Videos`文件夹下。

## 使用Kazam

### 安装

```shell
sudo apt install kazam
```
