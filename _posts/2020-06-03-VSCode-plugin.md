## Better Comments
### 功能
生成多种样式的注释
### 使用方法
```
// ! 警告类语句（红色）
// ? 疑问类语句（蓝色）
// TODO 待办类语句（橙色）
// * 高亮类语句（绿色）
// // 删除线语句（深灰色）
```
