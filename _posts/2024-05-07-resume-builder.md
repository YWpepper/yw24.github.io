## 相关概念区别

|  | Curriculum Vitae（CV） | Resume |
| :--- | :--- | :--- |
| 篇幅 | 长，一般2页 | 短，一般1页 |
| 内容 | 专业和学术成就 | 经验和技能 |
| 目的 | 申请教职、奖助学金、欧洲工作 | 申请美国工作 |
| 是否包含照片 | 是 | 否 |
